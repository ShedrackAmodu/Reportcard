System Functionalities

Comprehensive features designed for modern educational institutions

Offline-First Architecture

Complete offline functionality allowing teachers and students to input grades, view reports, and manage data without internet connectivity. All changes are automatically synchronized when connection is restored.

Manual Synchronization Control

Users have full control over when data synchronization occurs. Teachers can work offline for extended periods and choose when to sync their changes with the central server.

Customizable Grading Scales

Each school can define its own grading scales, letter grades, and assessment criteria. Support for various grading systems including percentage-based, letter grades, and custom scales.

Multi-Tenant Architecture

Complete data isolation between schools with dedicated configurations. Super administrators can manage multiple schools while maintaining strict data separation and security.

PDF Report Generation

Generate professional PDF report cards with customizable templates. Include school branding, student information, grades, and comments in downloadable and printable formats.

Cross-Platform Mobile App

Native mobile applications for iOS and Android devices. Teachers can manage grades on-the-go, and students can access their report cards from their smartphones.
